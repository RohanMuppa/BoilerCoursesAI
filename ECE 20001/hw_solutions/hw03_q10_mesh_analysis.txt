THIRD ANALYSIS (Supermesh approach)

If the dependent source is between Mesh 1 and Mesh 2 (shared branch), use supermesh.

Constraint equation.
Net current through source (upward) = I2 minus I1 = 3Io = 3I1
I2 minus I1 = 3I1
I2 = 4I1

Supermesh KVL (combining both meshes, avoiding the current source).
Write KVL for Mesh 1 and Mesh 2 separately, then add them.

Mesh 1 KVL.
12(I1) + 3(I1) + 6(I1 minus I2) + 24(I1) = 0

Mesh 2 KVL.
V_source + 6(I2 minus I1) minus 36 = 0

Adding Mesh 1 and Mesh 2.
12I1 + 3I1 + 6(I1 minus I2) + 24I1 + 6(I2 minus I1) minus 36 = 0
The 6 ohm terms cancel.
12I1 + 3I1 + 24I1 minus 36 = 0
39I1 = 36
I1 = 36/39 = 12/13 A

Therefore.
Io = I1 = 12/13 A approximately 0.923 A
I2 = 4I1 = 48/13 A approximately 3.69 A

Current through 6 ohm resistor.
I_6ohm = I1 minus I2 = 12/13 minus 48/13 = minus 36/13 A
Magnitude is 36/13 A approximately 2.77 A

Power absorbed by 6 ohm.
P = I squared times R = (36/13) squared times 6
P = (1296/169) times 6 = 7776/169 W approximately 46.01 W
