Q14 Boolean AlgebraSimplify X·Y'·Z + X·Y·Z + X'·Y·Z + X'·Y'·Z▶
Factor out Z from every term:
= Z · (XY' + XY + X'Y + X'Y')

Group first two and last two:
= Z · (X(Y'+Y) + X'(Y+Y'))

Apply complement theorem (Y+Y'=1):
= Z · (X·1 + X'·1)
= Z · (X + X')
= Z · 1
= Z
D. Z

Every possible combination of X and Y appears, so they cancel out completely, leaving only Z.

Q15 CMOSIdentify gate from CMOS circuit▶

CMOS Gate Identification Method:

1. Look at the PMOS network (connected to VDD, top)
2. Look at the NMOS network (connected to GND, bottom)
3. PMOS parallel = NMOS series (and vice versa)
4. PMOS determines when output is HIGH
5. NMOS determines when output is LOW
PMOS parallel → inputs ORed (any LOW input pulls HIGH)
PMOS series → inputs ANDed (all LOW inputs needed for HIGH)
NMOS parallel → inputs ORed (any HIGH input pulls LOW)
NMOS series → inputs ANDed (all HIGH inputs needed for LOW)

In the exam circuit: 3 PMOS in parallel, NMOS in series. Parallel PMOS means output HIGH when ANY input is 0. Series NMOS means output LOW when ALL inputs are 1. This is NAND.

D. NAND
Q16 CMOSNoise Margins NML and NMH▶

When Family A drives Family B (A output goes to B input):

NML = VIL(receiver) − VOL(driver) = VIL(B) − VOL(A)
NMH = VOH(driver) − VIH(receiver) = VOH(A) − VIH(B)
From the tables:
Family A: VOH_min = 2.7V, VOL_max = 0.3V
Family B: VIH_min = 1.0V, VIL_max = 0.5V

NML = VIL(B) − VOL(A) = 0.5 − 0.3 = 0.2V
NMH = VOH(A) − VIH(B) = 2.7 − 1.0 = 1.7V

Remember: Use OUTPUT specs from the DRIVER and INPUT specs from the RECEIVER. Low margin uses low specs (VOL, VIL). High margin uses high specs (VOH, VIH).

D. NML = 0.2V, NMH = 1.7V

Q18 CMOSRise time and propagation delay▶
Rise time (tr) = time from 10% to 90% of output swing
tpLH = time from 50% of input change to 50% of output change (low→high)
Using the 90/10 rule on the output waveform F:
1. Find where output starts rising
2. Mark 10% and 90% voltage points
3. tr = time between those two points

For tpLH:
1. Find the 50% point of the input transition
2. Find the 50% point of the output transition (going high)
3. tpLH = time difference

Read the grid carefully. Each grid square = 1ns based on the scale shown.

A. tr = 6.0ns, tpLH = 4.0ns
